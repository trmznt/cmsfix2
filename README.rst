CMSFix2
=======

CMSFix2 is a simple CMS rewritten from CMSFix <https://github.com/trmznt/cmsfix>_.
While the original was built using the Pyramid framework and the Rhombus add-on library,
CMSFix2 is built for the Litestar async framework and the Litestar-Pulse add-on library.

Installation
------------
To install CMSFix2, please execute the following command in your terminal:

.. code-block:: bash

   "${SHELL}" <(curl -L https://raw.githubusercontent.com/trmznt/cmsfix2/main/install.sh)
